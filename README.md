😊😢 Happy or Sad Image Classifier

This is a *Streamlit web app* that uses a *deep learning model(CNN)* (TensorFlow) to classify uploaded images as either Happy or Sad.

---

 🚀 Features
- Upload an image (JPG/PNG).
- The app predicts whether the image is **Happy** or **Sad**.
- Clean and interactive UI built with Streamlit.

---

 🛠️ Tech Stack
- Python 3
- TensorFlow / Keras
- NumPy
- Pillow (PIL)
- Streamlit
